Privacy Policy

Last updated: 2025

GroupNest is a privacy-first Chrome extension that helps users organize local files, notes, and links into workspaces.

Summary

GroupNest stores workspace metadata locally in your browser. Your actual files stay on your computer and are only read when you open, check, reconnect, import, or export them.

Data stored locally

GroupNest may store workspace names, section names, card names, notes, links, file metadata, local file references selected by you, workspace ordering, backup metadata, restore metadata, and local settings.

This data is stored locally in your browser using browser storage such as IndexedDB and Chrome storage.

File access

GroupNest only accesses files or folders that you explicitly select through your browser. GroupNest does not upload your files as part of its core local organization features.

GroupNest may read selected files when you open or preview a file, run a file health check, reconnect a file card, export a workspace as a ZIP, or import a GroupNest ZIP and restore files into a folder you choose.

ZIP export and restore

When you export a workspace as a ZIP, GroupNest reads the selected local files needed for that export and creates a ZIP file on your device.

When you import a GroupNest ZIP, GroupNest asks you to choose a local restore folder. Files from the ZIP are extracted into that folder. GroupNest then stores local references to those restored files.

Payments

GroupNest is currently available as an early access free release. No paid checkout is active at this time.

If paid premium features are introduced later, payment processing may be handled by a third-party payment provider. Payment-related data will be handled according to that provider's policies. GroupNest will update this policy before enabling paid checkout.

Data sharing

GroupNest does not sell user data.

GroupNest does not upload or share your local files, notes, links, workspace data, or file metadata as part of its core local organization features.

User control

You can use the Clear Data option inside GroupNest to remove local workspace metadata, notes, links, cards, and stored file references from your browser.

Clearing GroupNest data does not delete files from your computer.

Permissions

GroupNest uses Chrome's side panel to display the workspace manager and browser storage to store local workspace metadata and settings.
